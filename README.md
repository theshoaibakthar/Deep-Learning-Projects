# Deep Learning Projects

## Overview

This repository showcases a collection of Deep Learning projects, demonstrating practical implementation of various neural network architectures, advanced deep learning models, and cutting-edge deep learning techniques. Each project highlights different aspects of deep learning, from basic neural networks to advanced recurrent and convolutional architectures.

## Projects

### 1. [Simple RNN IMDb Sentiment Analysis](/Simple-RNN-IMDb/)

A comprehensive sentiment analysis system on IMDb movie reviews using Simple RNN:

- Implemented Simple RNN architecture for sequential text processing
- Built and trained deep learning model using Keras/TensorFlow
- Processed and vectorized the IMDb dataset for neural network input
- Developed prediction pipeline for sentiment classification

Skills demonstrated: RNN architecture, Sequence modeling, Text preprocessing, Deep learning frameworks, Model training and evaluation, Sentiment analysis
